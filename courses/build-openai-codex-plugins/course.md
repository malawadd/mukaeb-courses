# وصف الدورة

هذه الدورة موجهة للمطورين وفرق المنصات الداخلية الذين يريدون تحويل سير العمل المتكرر في `Codex` إلى **إضافة قابلة للتثبيت وإعادة الاستخدام** بدل الاعتماد على مجلدات skills محلية فقط.

## ستتعلم

- الفرق العملي بين `Skill` و`Plugin` و`MCP` و`Apps SDK`
- كيف تبني ملف `.codex-plugin/plugin.json` بطريقة صحيحة
- كيف تربط الإضافة مع `skills/` و`.mcp.json` و`.app.json`
- كيف تنشئ `marketplace.json` محليًا أو على مستوى المستودع
- كيف تختبر الإضافة داخل `Codex` وتفعّلها وتستدعيها صراحة أو ضمنيًا
- كيف تتعامل مع المصادقة وسياسات التثبيت والبيانات والصلاحيات
- كيف تجهز الإضافة للتوزيع داخل فريقك أو عبر Git-backed marketplace

## مخرجات الدورة

بنهاية المسار سيكون لديك Plugin متكامل لـ `Codex` باللغة العربية، مع manifest صحيح، skill حقيقي، وربط اختياري مع MCP أو Apps، بالإضافة إلى checklist واضحة للتثبيت والاختبار والتوزيع.
