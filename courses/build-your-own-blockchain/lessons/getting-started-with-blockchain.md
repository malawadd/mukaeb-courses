# ما هي البلوكشين؟

البلوكشين هي **دفتر سجلات رقمي** يتم نسخه على عدة أجهزة بدلًا من جهاز واحد.

{% callout title="الفكرة الأساسية" tone="tip" %}
فكر فيها كسجل مشترك بين عدة أطراف. كل نسخة ترى نفس التعديلات تقريبًا، وهذا هو جوهر الثقة الموزعة.
{% /callout %}

## لماذا هذا مهم؟

- يقلل الاعتماد على طرف مركزي واحد
- يصعّب التلاعب بالبيانات
- يجعل تتبع التغييرات أسهل

## كيف تتخيلها؟

تخيل دفترًا مشتركًا يملكه الفريق كله. أي تعديل جديد يراه الجميع، ولا يمكن تغيير القديم بسهولة دون أن يلاحظ الآخرون.

{% tabs %}
{% tab label="التشبيه اليومي" %}
تخيل أن كل فرد في الفريق يحمل نسخة من نفس الدفتر. عندما تُضاف صفحة جديدة، تصل النسخة إلى الجميع، فلا يوجد شخص واحد يستطيع إعادة كتابة الماضي بصمت.
{% /tab %}
{% tab label="المعنى التقني" %}
تقنيًا، نحن نتحدث عن **سجل موزع**، حيث كل كتلة تحمل بياناتها ومعها بصمة مرتبطة بما قبلها.
{% /tab %}
{% /tabs %}

{% accordion title="متى تصبح البلوكشين خيارًا جيدًا؟" %}
تكون مفيدة عندما تحتاج إلى:

- تاريخ تغييرات واضح
- مشاركة الثقة بين عدة أطراف
- صعوبة تعديل السجل السابق دون أثر
{% /accordion %}
