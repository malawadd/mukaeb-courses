# المطلوب

أنشئ مسودة مهارة جديدة لمهمة متكررة في مشروعك، ثم راجعها كأنك ستنشرها لفريق كامل.

## المطلوب داخل التحدي

- اختر مهمة حقيقية تستحق مهارة مستقلة
- اكتب `SKILL.md` يتضمن الهدف، حالات الاستخدام، وحالات عدم الاستخدام
- أضف workflow يمكن اختباره خطوة بخطوة
- راجع المهارة بقائمة تحقق قصيرة قبل اعتمادها

## معايير النجاح

- النطاق واضح وغير متضخم
- توجد إشارات تفعيل قابلة للملاحظة
- توجد حدود تمنع الاستخدام الخاطئ
- الخطوات قابلة للتنفيذ والتحقق
- المثال أو المخرجات المتوقعة واضحة

## امتداد اختياري

بعد إنهاء المهارة، حوّلها إلى محتوى داخل مستودع GitHub الخاص بالمحتوى، ثم ادفعها وشاهدها تدخل إلى المنصة كمسودة قبل النشر.
