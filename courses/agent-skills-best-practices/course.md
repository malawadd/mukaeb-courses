# وصف الدورة

هذه الدورة مخصصة لمن يريد تحويل خبرته التشغيلية إلى مهارات قابلة لإعادة الاستخدام داخل أنظمة الوكلاء.

## ستتعلم

- متى تحتاج إلى ملف `SKILL.md` أصلًا
- كيف تحدد نطاق المهارة بدل كتابة تعليمات عامة ومبهمة
- كيف تكتب أقسامًا تجعل الوكيل يعرف متى يستخدم المهارة ومتى لا يستخدمها
- كيف تراجع المهارة قبل نشرها وتمنع التضارب بينها وبين بقية التعليمات
- كيف تربط بين كتابة المحتوى في GitHub ورفع الدورة إلى المنصة

## مخرجات الدورة

بنهاية هذا المسار سيكون لديك قالب عملي لإنشاء مهارة جديدة، وقائمة مراجعة لتقييمها، وتحدي نهائي يحاكي سير العمل الحقيقي من المسودة إلى النشر.
